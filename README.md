# Tic-Tac-Toe Web Application

A responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Features

- Two Player Mode
- Play Against Computer (AI)
- Responsive Design
- Dark / Light Mode
- Scoreboard
- Winning Animation
- Local Storage
- Mobile Friendly

## Technologies Used

- HTML
- CSS
- JavaScript
